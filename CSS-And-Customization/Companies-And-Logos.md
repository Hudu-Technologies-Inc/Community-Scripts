# Companies and Logos

For users on (future) Hudu version 2.4.0, you'll find that Company Icons/Logos feature is available.

## Hiding Company Logos

If don't care for or wish to hide company logos, it's pretty simple to do so.

```css
.company-with-icon img {
    display: none !important;
}
.company__logo__blank {
    display: none !important;
}
```
